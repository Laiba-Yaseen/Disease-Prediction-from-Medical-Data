🩺 Disease Prediction using Machine Learning

📌 Project Overview

The Disease Prediction using Machine Learning project is a healthcare-focused application designed to predict disease conditions using medical data and Machine Learning techniques. Early disease prediction can assist healthcare professionals in making informed decisions and providing timely treatment.

This project utilizes the Breast Cancer Wisconsin Dataset and applies a Random Forest Classifier to classify tumors as either malignant or benign, demonstrating the practical application of Machine Learning in healthcare.


🎯 Project Objective

To develop a Machine Learning model capable of predicting disease conditions accurately using medical features and supporting early diagnosis in healthcare applications.


✨ Key Features

🩺 Medical disease prediction using Machine Learning

📊 Data preprocessing and feature selection

🌲 Random Forest classification model

📈 Model evaluation using multiple performance metrics

🔍 Prediction of malignant and benign conditions

💻 Implementation using Python and Google Colab

📊 Dataset

The project uses the Breast Cancer Wisconsin Dataset, which contains:

569 patient records

30 medical features

Binary classification:

Malignant

Benign

The dataset was preprocessed and divided into training and testing datasets for model development.
